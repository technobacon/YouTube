# WoW (Classic / Vanilla) Facts Ledger

Companion to `research/osrs-facts.md`, same two rules:

1. **Nothing gets scripted as a stat unless it's in here with a source, or freshly verified.** No numbers from memory.
2. **Every entry carries a source and a date checked.**

One difference from the OSRS ledger: vanilla (1.x, 2004–2006) is a *fixed target* — verified facts about it don't drift. Classic-era facts (2019+) and anything about current server status still need re-checking near publish. Prefer primary sources: warcraft.wiki.gg / classic.wowhead.com, Blizzard patch notes and newsposts, Warcraft Logs for clear-speed claims, archived community records (DKP systems, guild forums) for social-history claims.

## Format

```
| Fact | Value | Source | Date checked | Notes |
```
