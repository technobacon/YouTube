# Project N — Five Niche Videos (OSRS, exurb1a dial high)

Five new ideas, pitched 2026-07-07, on the owner's brief: **very niche, very specific OSRS topics**, end-to-end, exurb1a style. Where V/V2/W took systems (XP curves, death, classes, time), this batch takes *objects and rooms*: one cave, one dungeon, one cake, one beach custom, one party room. The wager is the exurb1a wager itself — the smaller the object, the bigger the thing it can open onto, provided the essay does the actual reading.

## Why these five

- **Genuinely niche.** No batch so far has gone below the system level. Each of these is the kind of topic a knowledgeable player has stood next to a thousand times without once considering essay-worthy — which is the Project V strategy note ("reframe a thing players think they understand") pushed to its smallest possible scale.
- **Zero overlap, checked object by object.** Tears of Guthix touches progression without touching the XP-curve video's territory (that one is about the price of strength; this one is about the subsidy of weakness). The Stronghold is security history, untouched anywhere. The rock cake/NMZ script flanks the drafted attention-tax material (one nod, no shared body sections) and stays off the minigame graveyard's incentives argument. "Spot taken" is emergent-norms sociology; the closest neighbor is W's raiding-institutions essay, in a different game and register. The Party Room script *argues with* the minigame graveyard video directly — its README-level relationship is "the exception that tests the rule," and the script says so on camera.
- **The callback web pays off hardest here.** Niche objects need the channel's established arguments as anchors: #1 leans on the XP-curve video, #3 on going-dry's variance material, #5 on the central-bank draft and the minigame graveyard. Mine, don't repeat — every callback is one sentence.
- **exurb1a fit is maximal at this scale.** The style's signature move (mundane object → cosmic register → tender landing) works *better* on a puddle than on an economy. This batch is the purest test of the pivot yet.

## Format spread

Cave tour / field trip / restaurant review / surveyor's report / weather broadcast. Nothing reused from V (museum, tour-documentary, split-screen, trial, funeral), W (anatomy, voyage, onboarding, briefing, lab report), or V2 (dig, appointment, support group, noir, nature film). One `[hold]` in the batch — #1's, on the tears falling unattended. One profanity per script in #2–#5; #1 runs clean (same deliberate exception as V2's bank essay — the register is better unbroken there).

## Suggested release order

1. **#3 (The Rock Cake)** — funniest cold, biggest named-content search pull (NMZ), the review format is instantly legible.
2. **#4 (Spot Taken)** — universal recognition ("taken?" has been typed by every viewer); comment section becomes a court of beach-law stories on its own.
3. **#2 (Stronghold of Security)** — nostalgia + history pull; strong shareability outside the core audience ("the dungeon that taught a generation about phishing").
4. **#1 (Tears of Guthix)** — the batch's heart; benefits from an audience already trusting the register.
5. **#5 (The Party Room)** — the quietest thesis; lands best once #1 has proven the tender register performs.

## Before recording any of these

Each script ends with a **Flags** section; log verified values into `research/osrs-facts.md` before cards. Niche topics are flag-dense by nature — the mechanics are obscure enough that even high-confidence memory needs the wiki pass. Nothing numeric appears in any script's prose except explicitly hedged approximations.
