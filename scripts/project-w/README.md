Five new ideas, pitched 2026-07-07. The channel's first full batch outside OSRS: **Classic-era World of Warcraft (2004–2006 design, 2019+ Classic where stated)**. Voice dial pushed harder toward `exurb1a.md` than Project V — this batch is the experiment the channel-context doc asked for ("one or two videos can push the voice harder"; here it's the house style, with the analysis still doing the argumentative work underneath).

Every file carries the full pipeline: **idea → engagement-mapped outline → SEO package → complete script → fact flags → §2 audit.**

## Why these five

- **One friction each.** The batch's shared spine is that vanilla's frictions were social and meaning-making machinery; each script takes exactly one (dependency, distance, organization, indifference, time) so the five never blur into each other. Cross-references between them are wired in where free (#2 nods at #1's warlock summons; #5 borrows #2's "we were fourteen" admission) — mine, don't repeat.
- **Zero overlap with the catalogue's WoW entry.** Video #2 on the channel ("Why OSRS Succeeded Where Classic WoW Struggled") argued what Classic+ should learn from OSRS. None of these five touch Classic+ or the OSRS comparison as a subject; they're design essays about vanilla on its own terms. That video is instead the *bridge* — proof ~9K of the channel's viewers already click WoW topics — and each description here cross-links it.
- **Every thesis passes the backlog bar.** A knowledgeable player can argue with each one: "the classes are broken on purpose," "the boat was good," "MC was easy," "you weren't the hero," "the experiment failed." All five reframe things Classic players think they already understand.
- **No PvP.** Carried over from Project V's brief. The honor-grind / rank 14 essay is an obvious sixth and stays in the backlog until the user says otherwise.

## The exurb1a dial, batch-wide

Project V pushed the register in two scripts; Project W runs it everywhere, per the pitch. Concretely: second-person address throughout, tonal whiplash at every landing (beauty → dumb joke → gut-punch), one precisely-placed profanity per script (never more), a story or parable dramatising each thesis instead of arguing it flat, and endings that resolve *emotionally, never intellectually* — every synthesis lands on an implication or an image, no recaps. The analysis layer stays rigorous (channel lane: arguments about design, not vibes); the voice is what carries it between the numbers. One `[hold]` exists in the whole batch (script #5, the mirror beat) — same rationing as Project V.

## Channel-fit notes (the honest risk)

- **Upside:** the WoW Classic search space is several times the size of OSRS's, the catalogue's WoW video already out-pulls most of the channel at ~8.8K, and the "reframe a known thing as a lie/secret" title shape that wins on this channel transfers cleanly ("Broken on Purpose," "You Were Never the Hero").
- **Risk:** channel identity is OSRS. Mitigations baked in: every title leads with vanilla/Classic keywords so search does the audience-finding, every description cross-links the OSRS↔WoW bridge video, and the essays' design-critic voice is continuous with the OSRS catalogue — same critic, new exhibit.

## Suggested release order

1. **#1 (Broken on Purpose)** — warmest entry point, safest facts, strongest "argue with me" thumbnail; establishes the batch's voice.
2. **#4 (You Were Never the Hero)** — the flagship provocation; broadest crossover to the general game-design audience.
3. **#5 (The 2019 Experiment)** — biggest evergreen search pull ("wow classic retrospective" traffic); benefits from two videos of warmed-up WoW audience.
4. **#3 (Raiding Was a Job)** — DKP/Molten Core keywords are durable; comment bait ("MC was easy") self-sustains.
5. **#2 (In Defense of the Boat)** — the format risk (a real-time, deliberately slow video); ship it once retention data from #1/#4 says how far the audience follows a conceit.

## Before recording any of these

Each script ends with a **Flags** section, and a new ledger exists at `research/wow-facts.md` — currently seeded with high-confidence-from-memory entries that all still need primary-source verification (Wowpedia/warcraft.wiki.gg, Blizzard patch notes, Warcraft Logs).
