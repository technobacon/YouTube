# Project V — Five Standalone Videos

Five new ideas, pitched 2026-07-07. No PvP, no Wilderness. Each one is a self-contained contrarian claim (the shape that pops per `channel-context.md`), and each uses a *different format* so the batch doesn't read as five reskins of the same essay.

Every file below carries the full pipeline: **idea → engagement-mapped outline → SEO package → complete script → fact flags → §2 audit.**

| # | File | One-line thesis | Format |
|---|------|-----------------|--------|
| 1 | `01-92-is-half-of-99.md` | The 2001 XP table — one exponential function, never re-tuned — is the best-designed thing in the game | **Museum object study** — the whole essay orbits a single number in a glass case |
| 2 | `02-minigame-graveyard.md` | Minigames die because OSRS taught players to price their hour, and fun is a currency the ledger doesn't accept | **Walking-tour documentary** — world-hopping through empty lobbies |

| 5 | `05-funeral-for-death.md` | The 2020 death rework buried the game's oldest teacher — and hardcore ironman is the player base buying the fear back | **Eulogy** — the video is a funeral service, movements and all |

## Why these five

- **Zero overlap** with drafted candidates (tick, attention tax, polls, GE bank, quests), the Sisyphus final, or the published catalogue (inventory, Moons ×3, WoW, Hueycoatl). The Slayer / going-dry / RuneLite seeds stay in the backlog untouched; #4 borrows adjacent territory to "going dry" but argues about *measurement*, not pity timers.
- **Every thesis passes the backlog bar:** a knowledgeable player could argue with each one, and each reframes a thing players think they already understand ("92 is half of 99" is the purest form of this — literally true, sounds insane).
- **Callbacks are wired in** where they're free: #1 nods at the central-bank video, #2 at the attention tax, #3 at Quest Helper. Mine, don't repeat.

## Format spread (the actual variety, not just topic variety)

Object study / travelogue / split-screen timeline / trial / funeral. Five different scaffolds for the same underlying spine (hook → tension → thesis → escalating body → turn → synthesis), so the channel tests which *container* holds retention best. Each script marks its voice dial; the trial and the funeral push the exurb1a register hardest, the museum piece stays the most grounded.

## Suggested release order

1. **#1 (92 Is Half of 99)** — safest facts (all ledger-verified), strongest thumbnail provocation, most "DPS is a lie"-shaped.
2. **#4 (Collection Log)** — high comment-bait potential; the trial format invites the audience to vote a verdict.
3. **#2 (Minigame Graveyard)** — evergreen search topic, big nostalgia pull.
4. **#3 (The Wiki)** — crossover appeal beyond OSRS (game-design audience).
5. **#5 (Funeral for Death)** — the format risk; schedule it once the trial's retention data says how far the audience follows a conceit.

## Before recording any of these

Each script ends with a **Flags** section. Everything flagged is either from memory (high confidence but unverified) or deliberately left vague in the prose. Verify flags against the wiki/newsposts and log them in `research/osrs-facts.md` before the on-screen cards get made. Ledger-sourced stats are cited inline in each file's header.
