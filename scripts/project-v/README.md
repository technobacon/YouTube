# Project V — Six Standalone Videos

The first five ideas were pitched 2026-07-07; #6 was added 2026-07-15. No PvP, no Wilderness. Each one is a self-contained contrarian claim (the shape that pops per `channel-context.md`), and each uses a *different format* so the batch doesn't read as reskins of the same essay.

Every file below carries the full pipeline: **idea → engagement-mapped outline → SEO package → complete script → fact flags → §2 audit.**

## Why these six

- **Zero thesis duplication.** #6 sits beside the no-dailies and attention-tax drafts without repeating either: "Nothing Is Waiting for You" argues that OSRS structurally permits absence; "The Attention Tax" maps the attention price of training methods; #6 stays player-side and asks how recovery, variety, and autonomy keep a chosen grind from becoming debt.
- **Every thesis passes the backlog bar:** a knowledgeable player could argue with each one, and each reframes a thing players think they already understand ("92 is half of 99" is the purest form of this — literally true, sounds insane).
- **Callbacks stay optional.** #1 nods at the central-bank video, #2 at the attention tax, #3 at Quest Helper. #6 can gesture once toward OSRS permitting absence, but its body belongs to player psychology and should stand alone.

## Format spread

Object study / travelogue / split-screen timeline / trial / funeral / four-exit intervention. Six different scaffolds for the same underlying spine (hook → tension → thesis → escalating body → turn → synthesis), so the channel tests which *container* holds retention best. The trial, funeral, and intervention push the voice hardest; the object study stays the most grounded.

## Suggested release order

1. **#1 (92 Is Half of 99)** — safest facts, strongest thumbnail provocation, most "DPS is a lie"-shaped.
2. **#4 (Collection Log)** — high comment-bait potential; the trial format invites the audience to vote a verdict.
3. **#6 (You Gotta Take a Break, Man)** — universal pain point and broad gaming crossover; publish away from the no-dailies essay so the packages never compete.
4. **#2 (Minigame Graveyard)** — evergreen search topic, big nostalgia pull.
5. **#3 (The Wiki)** — crossover appeal beyond OSRS into game design.
6. **#5 (Funeral for Death)** — the format risk; schedule it once the trial's retention data says how far the audience follows a conceit.

## Before recording any of these

Each script ends with a **Flags** section. Everything flagged is either from memory, derived, time-sensitive, or deliberately held below card-level confidence. Verify OSRS facts against the wiki/newsposts and external claims against their named primary sources, then log stable OSRS values in `research/osrs-facts.md` before the on-screen cards get made.
