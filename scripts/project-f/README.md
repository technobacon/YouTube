# Project F — Five Finance Essays (new-channel experiment, exurb1a dial high)

Five scripts, pitched 2026-07-08, for a **second-channel experiment**: video essays about money — personal finance, finance history, finance concepts — in the same exurb1a register the OSRS channel has been pivoting toward. Different game, same wager: take a thing people believe they understand (a credit score, a lottery ticket, a textbook fable) and show them the machine behind it, then let the machine open onto something about time, trust, or death.

Every file carries the full pipeline: **idea → engagement-mapped outline → complete script → music direction → 5-set SEO package → fact flags → §2 audit.**

## Niche research — what does well (2026-07-08)

Read before scheduling; this shaped the picks.

- **The story-driven collapse/fraud essay is the niche's proven engine.** Enron, FTX, Madoff, LTCM retellings reliably clear millions of views across many channels; the audience shows up for narrative catastrophe with real mechanics inside. Project F's history slot (#4, LTCM) plays this lane directly.
- **Faceless essay channels work in finance.** How Money Works (~840K subs, animated VO essays, contrarian myth-busting angles) and Patrick Boyle (~1.2M, deadpan wit + storytelling from an actual quant) prove both halves of the plan: no face needed, and dry humor over rigorous content is a differentiator, a crowded field of hype notwithstanding.
- **The winning title shape is the same one the OSRS channel already proved:** reframe a familiar thing as a lie/con/secret ("your credit score doesn't measure what you think," "the Einstein quote is fake"). Explainers describe; winners overturn.
- **Cost-of-living anxiety and "financial nihilism" are rising 2026 topics** — Gen Z "what's the point" money content is getting written about everywhere and barely has a definitive essay yet. #5 (lottery/hope) and #1 (compounding/time) sit on that nerve deliberately.
- **The gap in the niche is register.** Finance YouTube splits into hype (day-trading, crypto), explainer (Two Cents, Plain Bagel), and dry analysis (Boyle, Money & Macro). Almost nobody writes about money the way exurb1a writes about entropy — mortality-aware, tender, funny. That's the differentiator this batch tests. Closest existing analogue is none; nearest neighbors compete on information, not on feeling.
- **Length: the hits cluster 10–25 minutes.** These scripts target 12–15.

## Why these five

- **One per lane the owner named, with overlap on purpose:** #4 is pure finance history, #3 pure personal finance, #1/#2/#5 braid concept with the other two. If analytics later says one lane pops, the batch has a clean read on all three.
- **Every thesis passes the channel's proven bar:** someone financially literate could argue with each one. "The Einstein quote is fake" is checkable in one search. "The barter story never happened" contradicts every textbook. "The lottery is honestly priced" sounds indefensible until the essay lands it.
- **exurb1a fit is structural.** Compounding is exponential time (mortality). Money's origin is shared fiction and trust (loneliness/connection). The credit score is a Kafka number that watches you. LTCM is hubris versus an indifferent universe. The lottery is the retail price of hope. Each topic opens onto one of his wells without forcing it.
- **New channel = no callbacks.** The scripts stand fully alone; nothing references the OSRS catalogue. One deliberate exception in spirit: #1 reuses the "humans can't feel exponentials" insight the XP-curve video built, re-derived from scratch for a new audience — an idea the writer owns, told fresh, never cited.

## Format spread

Owner's manual / bedtime story / reference letter / crash investigation / guided meditation. Nothing reused from V (museum, tour, split-screen, trial, funeral), W (anatomy, voyage, onboarding, briefing, lab report), V2 (dig, appointment, support group, noir, nature film), or N (cave tour, field trip, restaurant review, surveyor's report, weather forecast). Two `[hold]`s in the batch — #4's (the recorder goes quiet) and #5's (eyes open). One profanity per script in #1, #3, #4; #2 and #5 run clean by design — the bedtime story and the meditation both die the moment the register breaks.

## Suggested release order

1. **#3 (Credit score)** — the most universally-felt grievance, strongest search volume ("credit score explained" is evergreen), the reveal (paying off a loan can hurt you) is proven comment-bait.
2. **#4 (LTCM)** — the collapse-story lane the niche rewards hardest; best crossover shareability; establishes the channel can do narrative.
3. **#1 (Compound interest)** — the fake-Einstein hook is a one-search-checkable provocation; benefits from an audience that has now seen the register twice.
4. **#5 (Lottery)** — the format risk; the meditation conceit needs viewers who already trust the voice.
5. **#2 (Barter myth)** — the quietest and most "essay" of the five; lands best once the channel has an audience that shows up for ideas rather than hooks.

## Before recording any of these

Each script ends with a **Flags** section. Ledger-style verification was done 2026-07-08 via web sources for the load-bearing stats (noted inline per file); everything else is flagged. Start a `research/finance-facts.md` ledger when the first of these goes to cards — same discipline as `research/osrs-facts.md`. Two apocryphal legends (#1's Einstein quote, #1's Fidelity dead-investors study) are used *as* legends — the scripts say on camera that they're fake; do not soften that into repeating them straight.
