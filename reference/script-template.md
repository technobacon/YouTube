# Script Template

Copy this into `scripts/drafts/<slug>.md` to start a new script. Delete the guidance in italics as you fill each part. The spine comes from `CLAUDE.md` §3; the voice comes from `exurb1a.md`; the `[VISUAL: ]` cues are paced per `CLAUDE.md` §4.5 — **one cue per beat (not per sentence), calm and slow by default**, so the edit stays cheap and on-register. Place a cue only where the image genuinely turns; assume every still gets a slow push/drift so long holds (10–20s+) are fine; save tight cutting for the hook and the rare emphasis spike. Every cue must also follow `CLAUDE.md` §4.5a: use obtainable stock, simple ChatGPT-generated assets, collectable game assets, easy in-game footage, stock video, text, or basic motion graphics; make the idea creative and the implementation obvious.

### Pre-delivery audit
- [ ] Scanned against `CLAUDE.md` §2a–2d (no "not X but Y", no one-word-question hooks, no rule-of-three padding, no banned vocab/filler, varied sentence rhythm).
- [ ] Every stat traces to `research/osrs-facts.md` or is flagged as unverified.
- [ ] Interpretation is labelled as interpretation, not stated as fact.
- [ ] Reads as a person talking: contractions, some fragments, no uniform sentence length.
- [ ] Starts with a **General visual style** that defines the video's achievable look and recurring motif.
- [ ] Every visual cue names or clearly implies an allowed source, a specific shot/composition, and simple motion; no cue requires advanced animation, difficult staging, or heavy compositing.
- [ ] Visual ideas are genuinely creative rather than generic, with complexity coming from metaphor, juxtaposition, framing, or timing.
